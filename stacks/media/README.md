# Media

**Libraries and streaming** for video, music, photos, and audiobooks — your files, your network.

## Stacks

- **[plex](plex/)** — Plex Media Server
- **[jellyfin](jellyfin/)** — Jellyfin (open media server)
- **[immich](immich/)** — Immich (self-hosted photos)
- **[audiobookshelf](audiobookshelf/)** — Audiobookshelf (audiobooks and podcasts)
- **[kavita](kavita/)** — Kavita (self-hosted ebooks, comics, and manga)
- **[open-ebooks](open-ebooks/)** — OpenBooks (IRC eBook search; see README for triage note vs “OpenEBOOKS”)
- **[open-audio](open-audio/)** — Navidrome (self-hosted music; folder name matches triage “Open Audio”)

If you add another media stack, put it in this folder and list it here.
