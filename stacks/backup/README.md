# Backup

Tools whose **primary job** is backing up or restoring data (scheduled jobs, encrypted copies, off-site sync).

## Stacks

- **[duplicati](duplicati/)** — Duplicati web UI for encrypted backups (great for paths mounted on the **server**).
- **[restic-rest-server](restic-rest-server/)** — **Restic** REST backend on the homelab; install **restic** on each **PC** for encrypted, deduplicated file backups to your server.

If you add another backup-focused stack, put it in this folder and list it here.
