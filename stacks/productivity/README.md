# Productivity

**Day-to-day personal or small-team tools:** files, feeds, calendars, and **note sync**—things that replace mainstream SaaS for daily work.

## Stacks

- **[nextcloud](nextcloud/)** — Nextcloud (files, calendar, contacts, etc.)
- **[syncthing](syncthing/)** — Syncthing (folder sync between your devices)
- **[paperless-ngx](paperless-ngx/)** — Paperless-ngx (PDF / scan archive, OCR, search, tags)
- **[freshrss](freshrss/)** — FreshRSS (self-hosted RSS)
- **[joplin-server](joplin-server/)** — Joplin Server (sync backend for Joplin notes; research **#245**)

If you add a stack here, list it above and follow **[docs/standards.md](../../docs/standards.md)** and **[CONTRIBUTING.md](../../CONTRIBUTING.md)**.
