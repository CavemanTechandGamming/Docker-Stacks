# Home & automation

**Local-first home automation, IoT, and household CCTV** — assistants, MQTT, Zigbee bridges, and **NVR** stacks that pair naturally with Home Assistant.

## Stacks

- **[home-assistant](home-assistant/)** — Home Assistant (home automation core)
- **[mosquitto](mosquitto/)** — Eclipse Mosquitto (MQTT broker for IoT and bridges)
- **[zigbee2mqtt](zigbee2mqtt/)** — Zigbee2MQTT (Zigbee coordinator → MQTT)
- **[frigate](frigate/)** — Frigate (local NVR / cameras + object detection)

If you add a stack here, list it above and follow **[docs/standards.md](../../docs/standards.md)** and **[CONTRIBUTING.md](../../CONTRIBUTING.md)**.
