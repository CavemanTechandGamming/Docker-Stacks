# AI

Self-hosted **machine learning and chat** workloads: run models and UIs on your own hardware instead of sending prompts to a vendor API.

## Stacks

- **[automatic1111-webui](automatic1111-webui/)** — AUTOMATIC1111 Stable Diffusion WebUI (prebuilt Docker image).
- **[comfyui](comfyui/)** — ComfyUI (ai-dock image, workspace bind mount).
- **[ollama-open-webui](ollama-open-webui/)** — Ollama plus Open WebUI for local LLM chat.
- **[openclaw](openclaw/)** — OpenClaw gateway (Control UI, channels, model providers).
- **[paperclip](paperclip/)** — Paperclip agent orchestration / web UI.

If you add a stack here, list it above and follow **[docs/standards.md](../../docs/standards.md)** and **[CONTRIBUTING.md](../../CONTRIBUTING.md)**.
